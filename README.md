## Sustainable-Brand-Survey
#### Overview
This repository contains a Jupyter notebook analyzes survey data on consumer perceptions and behaviors toward sustainable brands. The analysis includes data cleaning, frequency distributions, visualizations (e.g., bar charts), text categorization of qualitative responses, and statistical tests (e.g., chi-square for associations between demographics and sustainability perceptions).

Key insights:
Demographic breakdowns (country, education, income).
Purchase behaviors and barriers/drivers for sustainable products.
Categorization of perceptions (e.g., environmentally friendly, quality).
Statistical associations (e.g., age and education influence perceptions).
